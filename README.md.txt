This dashboard analyzes retail sales performance using transactional data across multiple regions and product categories.

Key metrics include Sales, Profit, Discount %, Returns, and Shipping Days.

The objective of this project is to identify:
• High-performing products and regions
• Impact of discounts on profitability
• Relationship between shipping delays and returns
• Opportunities to improve overall business performance

Tools Used:
- Tableau Public
- Excel / CSV dataset

This project demonstrates skills in data visualization, KPI analysis, and business insight generation.
